# Personal Portfolio Website – Project Documentation
 1. Project Overview :
   This project is a **Personal Portfolio Website** designed and developed to showcase personal information, skills, projects, and contact details in a **modern, interactive, and user‑friendly** way. The website    follows a **classy neon theme** with smooth animations and responsive design principles.

2. Objectives:
* Create a professional online presence
* Showcase skills, tools, and projects visually
* Provide easy contact options with a working contact form
* Use modern UI/UX practices
* Ensure responsiveness and smooth animations

3. Technologies Used:
   Frontend Technologies:
   HTML5 – Structure of the website
   CSS3 – Styling, layout, animations, neon effects
   JavaScript (ES6) – Interactivity and dynamic behavior
   Libraries & Tools:
   Font Awesome – Icons for skills and social media
   EmailJS – Contact form email functionality
   VS Code – Code editor
   Git & GitHub** – Version control
